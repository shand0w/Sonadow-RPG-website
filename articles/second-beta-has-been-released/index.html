Pixel Zone - Second beta has been releassed

After a few months of hard work, I was able to release the sceond beta of Pixel Zone "v0.9.2beta-2". It was built on Godot Engine 3.2beta2.


What's new?

- Fixed bugs with AI
- Leaderboard has been disabled (because a lot of bugs),
and it will be automatically enabled when it will be done.
- Added the ability to add your own music (in Level Editor) from the disk in addition to those to be selected from the list.
The file path cannot be changed. Then the level will not load. The game is built with 59a6018 commit. New links are on the download page.
