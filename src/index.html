  • Pixel Zone

  • How does modifications system works

    Resources from modifications are loaded 'next to' game resources. In this way you can use the game's resources and one modification at the same time. If the resources have the same name and path, the game will not load them correctly. Modification for the scene of the main game menu (in the code called GUI.tscn) is the scene with the path 'res: //mod.tscn', but it is not required to load modification.

  • Game crashes or it is not responding

    At some moments (pressing the button with the name of the selected character, or registration / login to the game) the game seems to stand still. In fact, the game loads the required resources at such times for example files, graphics or scripts.

    Sometimes in this case, Windows may decide that the application is not responding - this is normal. In such cases, however, you should not click as much as possible, because it will lead to the suspension or shutdown of the game due to too many very demanding tasks (selection of characters)
    or due to an error sending data to user authorization servers. Such data can only be sent individually, so please be patient in this case.
    It may be also effect of an bug. If you want, you can sumbit me bugs. Read more at home page

  • How do I make my own theme for Pixel Zone

    1. You must download the source code of the project to make it easier for you to create a theme.

    2. The project must be imported to Godot Engine (version 3.2, 3.2.x or higher)

    3. Choose one of the themes available in the project code and edit it, or create a new theme.

    Instructions on how to create a theme can be found in the godot documentation - here: https://docs.godotengine.org/en/3.1/tutorials/gui/gui_skinning.html

    4. Requirements. The theme can be located only from one file - < name > .tres

    5. Game must be restarted after deleting adding or modifying extsting theme in 'Pixel Zone' directory

  • Where to place a theme file so that the game will detect it

    Open the 'Documents' folder on your computer and find the 'Pixel Zone' folder there.

    There, paste the file with the theme and run the game to load it during the game.

  • The 'Pixel Zone' folder does not exist in the 'Documents' location

    If you did not start the game for the first time - do it.

    The game will then create the appropriate folders and configuration files If the folder is deleted the first time you start the game.
    When you start the game, it will create it again with all subfolders and configuration files.

  • Level Editor in Pixel Zone

  • How do I add an Tile

    Press 'SELECT TILES' button, then select a tile and press space. When you want to place tile again press space.

  • How do I add an background

    Press 'Add...' button, press 'Add background' then select a background do you want to use.

  • How do I add an music

    Press 'Add...' button, press 'Add music' then select a music do you want to use.

  • How do I save a level

    Press 'SAVE' button in left, top corner. Set a name for a level, press SAVE, and wait minute, while game is saving your game/

  • How do I run a level

    In main menu press 'Play' and select a character.
    when you move to the level selection screen, click the "<=" button on the right side
    of the screen and select the level, you created, from the list.

  • How to scale or rotate an object

    Select a tile, then press the space bar twice. Press the 'OPTIONS' button at the top of the screen and set the desired values.
    The first option - scale in the x axis;
    The second option - the scale on the y axis;
    Third option - snapping in the x axis;
    The fourth option - snapping in the y axis; Fifth option - Rotation degrees.

  • How to make Custom Tile?

    1. Open Custom Tile Creator via Level Editor

    2. Add graphics

    3. Add collision

    4. Scale collision and graphics if it is needed

    5. Export it

    6. Open Level Editor, You will see tiles, you made at the bottom of Tiles List.

  • How do i change tile in level editor

    1. Select the button that should add an instance of the tile you prepared earlier
    and find its 'pressed' method. It should look something like this:
    func _on_Tile2_pa_11_pressed():
    node = preload("res://LevelEditor/tiles/PixelAdventure/tile11.tscn").instance()
    node.set_name('tile_pa' + str(number))
    root.add_child(node)
    node.set_owner(root)
    $SetCurrentTile.hide()

    2. In a line:
    node = preload("res://LevelEditor/tiles/PixelAdventure/tile11.tscn").instance()
    change the contents of 'preload()',
    to the path that leads to the scene that the tile is selected for.

    IMPORTANT! THE PATH MUST BE A STRING

    3.Don't forget to change the texture 'normal'
    (in the properties of the texture button you choose) to one,
    that will be tile.

  • Problems with the game

  • Game don't want to run.

    On Linux:
    Go to Game's installation directory.
    Open properties tab and select 'Allow executing file as a program'.
    Check if there is .pck file. If no, download installer and execute it again.


    On Windows and Mac OS X:
    Go to Game's installation directory.
    Check if there is .pck file. If no, download installer and execute it again.

  • Game is not sending my score to leaderboard.
    (There are Null names in leaderboard instead of my name)

    1. Check Your WiFI connection status
    2. Check if you are logged in
    3. Restart game.
    Game is saving your score also in encrypted file on PC,
    so when next time you will launch game, it will send score loaded
    from file to server, but only if you will be logged into Your Account.
